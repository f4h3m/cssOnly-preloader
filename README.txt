Css Only Preload plugin for WordPress can set any css only animation as your website Preloader.

== Description ==

CSS Only Preload plugin for WordPress can set any CSS only animation as your website Preloader.
You just need to paste the HTML and the CSS code of the Preloader

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload the plugin files to the `/wp-content/plugins/css-only-preloader` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the Settings->Css Preloader screen to configure the plugin
